March 9, 2010: Ehcache 2.0 Released

In the last six months, we have invested tens of thousands of person-hours into improving Ehcache, adding features and improving performance with the Terracotta Server Array. This latest release represents the culmination of all of that work. We are pleased to announce Ehcache 2.0.

New Features Include:

  • Hibernate 3.3 SPI caching plugin ›
  • Write-behind ›
  • JTA support ›
  • Ultra-fast bulk-loading API ›
  • Better scale-out with Terracotta's distributed cache engine ›
  • Dynamic, runtime cache configuration ›

If you already use Ehcache, you will love Ehcache 2.0, which is fully backward compatible. If you don't use Ehcache yet, take it for a spin and find out why it's the industry-standard Java cache.

—Greg Luck, Founder of Ehcache
