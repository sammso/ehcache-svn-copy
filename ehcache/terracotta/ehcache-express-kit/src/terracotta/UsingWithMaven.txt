Using Ehcache and Terracotta with Maven
=======================================

Using Ehcache with Terracotta Server is easy with the Terracotta tc-maven-plugin.

Integration
============

To start and stop the server pre and post integration tests, add the following to your pom.xml:

<pluginRepositories>
    <pluginRepository>
        <id>terracotta-snapshots</id>
        <url>http://www.terracotta.org/download/reflector/maven2</url>
        <releases>
            <enabled>true</enabled>
        </releases>
        <snapshots>
            <enabled>true</enabled>
        </snapshots>
    </pluginRepository>
</pluginRepositories>


<plugin>
    <groupId>org.terracotta.maven.plugins</groupId>
    <artifactId>tc-maven-plugin</artifactId>
    <version>1.5.1</version>
    <executions>
        <execution>
            <id>run-integration</id>
            <phase>pre-integration-test</phase>
            <goals>
                <goal>run-integration</goal>
            </goals>
        </execution>
        <execution>
            <id>terminate-integration</id>
            <phase>post-integration-test</phase>
            <goals>
                <goal>terminate-integration</goal>
            </goals>
        </execution>
    </executions>
</plugin>



Interactive Use
===============

To use interactively, with the pom configured, use the following commands:

mvn tc:start to start

and mvn tc:stop to stop


More Information
================

See http://forge.terracotta.org/releases/projects/tc-maven-plugin for complete documentation.
