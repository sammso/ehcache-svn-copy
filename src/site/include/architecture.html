Hibernate Caching Provider

Full featured Hibernate caching provider supporting all cache strategies.

JMS Replication

The Ehcache jmsreplication module lets organisations with a message queue investment leverage it for replicating caches.

RMI Replication

RMI-based replication in clusters. Does not support coherency or transactions.

Ehcache Core

The core API. Most usages of Ehcache interact with CacheManager, Cache and Elements, contained in this API.

JGroups Replication

JGroups based replication in clusters. Does not support coherency or transactions.

Web Caching

Ehcache provides a set of general purpose web caching filters.

Cache Server

A Cache Server, available as a WAR for most web containers, or as a standalone server. The Cache Server has two APIs: RESTful resource oriented, and SOAP. Both support clients in any programming language.

Terracotta Distributed Caching

Distribution with the Terracotta Server Array ("TSA") is the preferred distribution mechanism. It provides coherency, JTA, HA, scale and high performance. It is available as open source or with additional features in the Ehcache EX and FX product editions. This requires a Terracotta server which is included in the Ehcache download, since 2.1.

JCache Support

Ehcache provides a preview implementation of JSR107 via the net.sf.cache.jcache package.

OpenJPA Caching Provider

A cache provider for the OpenJPA ORM.

DAO/JDBC Caching

Interaction with a database directly rather than via ORM remains popular. Ehcache's core API is useful for caching these accesses which are documented in this API.
