This package contains the ehcache server code. The SOAP and RESTful Web Service implementations are in subpackages.

Both are packaged and deployed in the ehcache-server.war.

The server has a context root of /ehcache.

The full EhcacheWebServiceEndpoint path is /ehcache/soap/EhcacheWebServiceEndpoint.

The full Ehcache RESTful Web Services path is /ehcache/rest/ with resources using a URITemplate.
